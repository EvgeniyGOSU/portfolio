⚙️ CI/CD Pipeline

┌─────────────────────────────────────────────────────────────────────────────┐
│                           git push (main / test)                            │
└─────────────────────────────────┬───────────────────────────────────────────┘
                                  ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│  Stage 1: test                                                              │
│  ├── go mod download                                                        │
│  └── go test -v ./...                                                       │
└─────────────────────────────────┬───────────────────────────────────────────┘
                                  ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│  Stage 2: build                                                             │
│  ├── docker build -t weather-bot:latest                                     │
│  └── docker push 192.168.181.134:30500/weather-bot:latest                   │
└─────────────────────────────────┬───────────────────────────────────────────┘
                                  ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│  Stage 3: deploy                                                            │
│  ├── kubectl apply -f k8s/                                                  │
│  └── kubectl rollout restart deployment/weather-bot                         │
└─────────────────────────────────────────────────────────────────────────────┘

⏱️ Весь процесс после git push занимает ~2 минуты

Инфраструктура

  • Kubernetes (Kubespray) — развёртывание кластера
  • Flannel CNI — сетевая подсистема
  • Helm — управление пакетами
  • cert-manager — SSL сертификаты
  • Local Registry — приватное хранилище образов
🔄

CI/CD

  • Gitea Actions — CI/CD пайплайны
  • Gitea Runner (K8s executor) — выполнение задач
  • Автоматический деплой при git push
  • GitOps — всё в репозитории
🔐

Сеть и безопасность

  • WireGuard VPN — туннель VPS → кластер
  • Hysteria2 прокси — обход блокировок
  • Let's Encrypt SSL — HTTPS сертификаты
  • NGINX reverse proxy — единая точка входа
📊

Мониторинг

  • Prometheus — сбор метрик
  • Grafana — визуализация
  • Кастомные метрики бота — requests, errors, latency
  • kube-prometheus-stack — мониторинг кластера
🤖

Weather Bot

  • Go 1.23 — высокопроизводительный бот
  • Telegram Bot API — интеграция с Telegram
  • WeatherAPI — данные о погоде
