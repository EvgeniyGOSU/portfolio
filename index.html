Обо мне

DevOps инженер с опытом построения production-инфраструктуры с нуля. Специализируюсь на Kubernetes, CI/CD, облачных технологиях и автоматизации.

🚀

Ключевой проект

Telegram-бот погоды с полноценной инфраструктурой: Kubernetes кластер, CI/CD, мониторинг, HTTPS и доступ из интернета через WireGuard.

Go Kubernetes Gitea CI/CD Prometheus Grafana

Метрики в Grafana

  • weather_bot_requests_total — запросы по командам
  • weather_bot_errors_total — ошибки API
  • weather_bot_request_duration_seconds — время ответа
⚙️

CI/CD Pipeline

  • ✅ test — go test
  • ✅ build — docker build + push
  • ✅ deploy — kubectl apply + rollout

Весь процесс после git push занимает ~2 минуты.
