☁️ DevOps / Backend Pet Project

Weather Scanner Bot

Telegram-бот для погоды с полным CI/CD пайплайном, деплоем в Kubernetes и мониторингом Prometheus + Grafana

🤖

Telegram бот

Go 1.23, Telegram Bot API, WeatherAPI. Команды /start и /weather с ответом о погоде в любом городе.

Go REST API
🐳

Контейнеризация

Multi-stage Dockerfile, образ 9MB, приватный GitLab Container Registry.

Docker GitLab Registry
☸️

Оркестрация K8s

K3s-кластер: Deployment, Service, ConfigMap, Secrets, Sidecar-контейнер.

K3s Kubernetes
🔄

CI/CD GitLab

Три стадии: test → build → deploy. При каждом git push бот обновляется автоматически.

GitLab CI/CD GitLab Runner (K8s executor)
🔐

Обход блокировок

Hysteria2 sidecar-контейнер с SOCKS5-прокси. Трафик идёт через VPS в Нидерландах.

Hysteria2 SOCKS5 TLS/QUIC
📊

Мониторинг

Prometheus собирает метрики (запросы, ошибки, latency), Grafana визуализирует.

Prometheus Grafana ServiceMonitor

Метрики в Grafana

  • weather_bot_requests_total — количество команд
  • weather_bot_errors_total — ошибки API/формата
  • weather_bot_request_duration_seconds — время ответа
  • weather_bot_active_users — активные пользователи
⚙️

GitLab CI/CD Pipeline

  • ✅ test — go mod download + go test
  • ✅ build — docker build + push в Registry
  • ✅ deploy — kubectl set image + rollout status

Весь процесс после git push занимает ~2 минуты.
